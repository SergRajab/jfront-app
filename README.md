## Requirements

- [Node.js](https://nodejs.org/en/download/package-manager/)
- [jfront-components](https://github.com/Jepria/jfront-components)

---

## Installation

```sh
yarn
```

or

```sh
npm i
```

## Running


### Build

```sh
npm run build
```

### Start

```sh
npm start
```
